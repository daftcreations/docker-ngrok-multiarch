# docker-ngrok-multiarch

 ngrok multiarch scratch container image

supported plateforms

- linux/amd64
- linux/arm64
- linux/arm/v6
- linux/arm/v7
- linux/386
- linux/mips64le
- linux/mips64
- linux/ppc64le

## Usage

```docker run --rm -it pratikimprowise/nginx http 80```
